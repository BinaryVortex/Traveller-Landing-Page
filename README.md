## About
Traveller is a single-page landing layout implemented with semantic HTML and modern CSS. It is intended as a lightweight starter for travel-related landing pages, promotional sites, or learning projects focused on layout and styling.

## Features
- Simple, easy-to-read HTML structure
- Styles written in plain CSS (no frameworks)
- Focused on a clear, visual hero/landing experience
- Easy to customise and extend

- CSS3
